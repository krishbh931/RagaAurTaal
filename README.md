# Raga Aur Taal Website

A static starter website for **Raga Aur Taal** — Indian classical music, bhajans, devotional learning, recordings, and Krish's ABGMVM journey.

## Files

- `index.html` — site content and page sections
- `styles.css` — full visual theme
- `script.js` — small reveal animation and footer year
- `assets/logo.jpeg` — logo image

## Run locally

Open `index.html` in a browser.

## Push to GitHub

```bash
git init
git add .
git commit -m "Initial Raga Aur Taal website"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/raga-aur-taal.git
git push -u origin main
```

## Publish with GitHub Pages

1. Go to the repo on GitHub.
2. Open **Settings** → **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select `main` and `/root`.
5. Save.

GitHub will provide a public website URL after deployment.
